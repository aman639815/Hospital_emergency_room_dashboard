
# 🏥 Hospital Emergency Room Dashboard

## 📌 Overview
This project focuses on analyzing hospital emergency room data to improve patient management and reduce waiting time.

## 🛠 Tools Used
- Microsoft Excel  
- Power Query  
- Data Visualization  

## 📊 Key Insights
- Total Patients: 451  
- Average Waiting Time: 35.47 mins  
- Patient Satisfaction Score: 4.59  
- Peak load observed in specific departments  

## 📁 Files Included
- Hospital Emergency raw data.csv → Dataset  
- Hospital_ER_Dashboard.xlsx → Dashboard file
