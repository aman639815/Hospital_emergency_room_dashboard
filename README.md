
# 🏥 Hospital Emergency Room Dashboard

## 📌 Overview
This project focuses on analyzing hospital emergency room data to improve patient management and reduce waiting time.

## 🛠 Tools Used
- Microsoft Excel  
- Power Query  
- Data Visualization  

## 📊 Key Insights
- Patient volume increased in 2024, showing higher emergency demand
- Average wait time (~35 mins) indicates scope for improvement
- 53% admission rate reflects a significant number of critical cases
- 39% delays highlight operational inefficiencies


## 📁 Files Included
- Hospital Emergency raw data.csv → Dataset  
- Hospital_ER_Dashboard.xlsx → Dashboard file
